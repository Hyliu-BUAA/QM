# 1. `SNS-MP2` 计算实例
## 1.1. 输入文件

<font color="73DB90" size="4">

1. Here's probably the simplest possible input file. It computes the interaction energy between two helium atoms seperated by two angstroms.

</font>

```
molecule {
He  0   0   0
--
He  2   0   0
}

import snsmp2
energy('sns-mp2')
```

## 1.2. 运行计算程序
```shell
linux > psi4 input.dat output.dat
```
